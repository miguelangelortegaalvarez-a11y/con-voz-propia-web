# Con voz propia — web pública

Web estática de **Con voz propia**: preparación de oposiciones de Audición y Lenguaje (Andalucía).

- `index.html` — landing (método, sobre mí, la oposición, contacto)
- `demo.html` — demo pública de la zona de alumnas (datos ficticios)
- `blog/` — blog SEO (índice + artículos; `_plantilla-articulo.html` para posts nuevos)
- `legal/` — aviso legal y privacidad
- `css/blog.css` — estilos compartidos de blog y legales
- `assets/og-image.png` — imagen para compartir en redes

**Importante:** este repo contiene SOLO la web pública. El material real de preparación
(temario, presentaciones, supuestos) vive fuera y nunca debe subirse aquí.

Hosting: GitHub Pages. Dominio previsto: `convozpropia.es`.
